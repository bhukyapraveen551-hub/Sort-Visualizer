bubble sort
Bubble Sort

Bubble Sort compares adjacent elements and swaps them if they are in the wrong order. With each full pass, the largest unsorted element "bubbles" to the end of the array. This continues until the array is sorted.

  • Best case: O(n)
  • Average case: O(n²)
  • Worst case: O(n²)
  • Space: O(1)
Selectionsort
Insertion Sort

Insertion Sort builds the sorted array one item at a time. For each new element, it scans the sorted part from right to left and inserts the element at the correct position by shifting others as needed.

  • Best case: O(n)
  • Average case: O(n²)
  • Worst case: O(n²)
  • Space: O(1)
merge sort
Merge Sort

Merge Sort uses a divide-and-conquer approach. It splits the array into halves, recursively sorts them, and merges the sorted halves. The merge step ensures elements are placed in the correct order.

  • Best case: O(n log n)
  • Average case: O(n log n)
  • Worst case: O(n log n)
  • Space: O(n)
quick sort
Quick Sort

Quick Sort selects a pivot element(represented in red) and partitions the array so that elements less than the pivot go to the left and greater ones go to the right. It then recursively sorts the partitions. Good pivot choice is key to performance.

  • Best case: O(n log n)
  • Average case: O(n log n)
  • Worst case: O(n²)
  • Space: O(log n)
